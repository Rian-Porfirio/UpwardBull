# UpwardBull

## Descrição
UpwardBull é um sistema de compras para todas as empresas, desenvolvido com React e Vite. Este projeto visa oferecer uma solução eficiente e interativa para o controle financeiro das empresas que o utilizam

## Funcionalidades
- **Gerenciamento de Produtos:** Adicionar, editar e excluir produtos facilmente.
- **Gerenciamento de cotações:** Monitorar e atualizar as cotações de produtos.
- **Gerenciamento de fornecedores:** Adicionar fornecedores que vendem os produtos necessitados.
- **Gerenciamento de contatos** Adicionar contatos de respectivos fornecedores.

## Tecnologias Utilizadas
- **Frontend:** React
- **Ferramentas de Desenvolvimento:** Vite
- **Estilização:** Tailwind CSS e DaisyUI

## Como Começar
1. Clone o repositório:
   ```sh
   git clone https://github.com/Rian-Porfirio/UpwardBull.git
2- Crie um arquivo .env para adicionar as variáveis de ambiente que serão utilizadas para o consumo dos serviços do firebase.
    VITE_FIREBASE_API_KEY
    VITE_FIREBASE_AUTH_DOMAIN
    VITE_FIREBASE_PROJECT_ID
    VITE_FIREBASE_STORAGE_BUCKET
    VITE_FIREBASE_MESSAGING_SENDER_ID
    VITE_FIREBASE_APP_ID
    
3- npm run dev para executar o sistema.
